# 📘 Docker Introduction

## What is Docker?
Docker is a containerization platform that allows developers to package applications with all dependencies into lightweight containers.

---

## Why Docker?

- Works on any system
- Eliminates "It works on my machine" problem
- Fast deployment
- Lightweight compared to Virtual Machines

---

## Virtual Machine vs Container

| Virtual Machine | Docker Container |
|-----------------|------------------|
| Heavy | Lightweight |
| Has full OS | Shares host OS |
| Slow startup | Fast startup |

---

## Key Components

- Docker Engine
- Docker Images
- Docker Containers
- Docker Hub
